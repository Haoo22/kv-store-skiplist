系统结构速览

  • 协议层 纯文本命令，支持 PING / PUT / GET / DEL / SCAN / QUIT
  • 核心网络 单线程 Reactor + 非阻塞 socket + epoll 事件循环
  • 存储引擎 跳表索引负责有序读写与范围扫描
  • 持久化 WAL 先写日志再更新内存，支持重启恢复

答辩主张

  • 主线版本保留单线程 Reactor，因为线程池实验出现端到端退化。
  • 项目同时具备协议验证、交互客户端、benchmark 与对比实验链路。
  • 展示重点不是“做了并发”，而是“做了实现并用实验验证取舍”。
  • 页面中的动态 QPS 与延迟用于答辩展示，正式结论以 README 中的 benchmark 结果为准。
